# 说明
此项目用于备份个人博客的文章内容。博客内阅读体验更加，[点击阅读(http://blog.airfunc.com)](http://blog.airfunc.com/)。

文章内容将以每周至少一篇的进度进行更新（自己立的flag）。内容将会以技术分享为主，好物分享（优质书籍、github 开源项目、优秀文章等）为辅，敬请期待。

对于文章中的问题可以通过邮件、issues、博客评论提出，我会在第一时间回复😄。
